# Benachrichtigung nach dem Speichervorgang (Dialogfeld) (Visual Database Tools)
Dieses Dialogfeld wird angezeigt, wenn beim Versuch, eine Tabelle zu speichern, Fehler auftreten. In diesem Dialogfeld werden die erfolgreich in der Datenbank gespeicherten Objekte, die nicht gespeicherten Objekte und die aufgetretenen Fehler aufgeführt.  
  
## Optionen  
**Textdatei speichern**  
Anzeigen der **Speichern** Dialogfeld, dem Sie aufgefordert werden, eines Speicherort zum Speichern einer Textdatei mit den Fehlerinformationen. In dieser Datei werden sowohl die erfolgreich gespeicherten Datenbankänderungen als auch die Änderungen aufgezeichnet, die aufgrund von Fehlern nicht gespeichert werden konnten.
