# Anzeigen von Diagrammeigenschaften (Visual Database Tools)
Sie können Eigenschaften für ein Datenbankdiagramm im Eigenschaftenfenster anzeigen.  
  
### So zeigen Sie Datenbankdiagrammeigenschaften im Eigenschaftenfenster an  
  
1.  Öffnen Sie den Datenbankdiagramm-Designer.  
  
2.  Stellen Sie sicher, dass keine Objekte im Datenbankdiagramm-Designer ausgewählt sind. Klicken Sie dazu auf eine Stelle, an der sich kein Objekt befindet.  
  
3.  Aus der **Ansicht** Menü klicken Sie auf **Eigenschaftenfenster**.  
  
Die Eigenschaften des Diagramms sind nun im der **Eigenschaften** Fenster.
