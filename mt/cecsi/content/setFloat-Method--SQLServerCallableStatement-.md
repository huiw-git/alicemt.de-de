# setFloat-Methode (SQLServerCallableStatement)
    
## setFloat\-Methode \(SQLServerCallableStatement\)  
 Legt den angegebenen Parameter auf den angegebenen Wert vom Typ **float** fest.  
  
## Syntax  
  
```  
  
public void setFloat(java.lang.String sCol,  
                     float f)  
```  
  
#### Parameter  
 *sCol*  
  
 Ein **String** mit dem Parameternamen.  
  
 *f*  
  
 Ein Wert vom Typ **float** .  
  
## Ausnahmen  
 [SQLServerException](../content/SQLServerException-Class.md)  
  
## Hinweise  
 Diese setFloat\-Methode wird von der setFloat\-Methode in der java.sql.CallableStatement\-Schnittstelle angegeben.
