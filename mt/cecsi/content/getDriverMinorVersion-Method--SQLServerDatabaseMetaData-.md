# getDriverMinorVersion-Methode (SQLServerDatabaseMetaData)
    
## getDriverMinorVersion\-Methode \(SQLServerDatabaseMetaData\)  
 Ruft die Nebenversionsnummer dieses JDBC\-Treibers ab.  
  
## Syntax  
  
```  
  
public int getDriverMinorVersion()  
```
