# isNamed-Methode (SQLServerSavepoint)
    
## isNamed\-Methode \(SQLServerSavepoint\)  
 Überprüft, ob dem Sicherungspunkt ein Name zugewiesen wurde.  
  
## Syntax  
  
```  
  
public boolean isNamed()  
```  
  
## Rückgabewert  
 **true**, wenn der Speicherpunkt benannt ist. Andernfalls wird **false** verwendet.
