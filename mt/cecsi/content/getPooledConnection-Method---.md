# getPooledConnection-Methode ()
    
## getPooledConnection\-Methode \(\)  
 Stellt eine physische Datenbankverbindung her, die als Poolverbindung verwendet werden kann.  
  
## Syntax  
  
```  
  
public javax.sql.PooledConnection getPooledConnection()  
```
