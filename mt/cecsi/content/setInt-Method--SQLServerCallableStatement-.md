# setInt-Methode (SQLServerCallableStatement)
    
## setInt\-Methode \(SQLServerCallableStatement\)  
 Legt den angegebenen Parameter auf den angegebenen Wert vom Typ **int** fest.  
  
## Syntax  
  
```  
  
public void setInt(java.lang.String sCol,  
                   int i)  
```  
  
#### Parameter  
 *sCol*  
  
 Ein **String** mit dem Parameternamen.  
  
 *i*  
  
 Ein **int** \-Wert.  
  
## Ausnahmen  
 [SQLServerException](../content/SQLServerException-Class.md)  
  
## Hinweise  
 Diese setInt\-Methode wird von der setInt\-Methode in der java.sql.CallableStatement\-Schnittstelle angegeben.
