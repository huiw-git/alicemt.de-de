# SQLServerBlob-Klasse
  Stellt ein BLOB \(Binary Large Object\) dar.  
  
 **Paket:** com.microsoft.sqlserver.jdbc  
  
 **Erweitert:** java.lang.Object  
  
 **Implementiert:** java.sql.Blob, java.io.Serializable  
  
## Syntax  
  
```  
  
public class SQLServerBlob  
```  
  
## Hinweise
