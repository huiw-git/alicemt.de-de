# Umbenennen von Abfragen (Visual Database Tools)
Sie können jede Abfrage umbenennen, die aktuell nicht zur Bearbeitung geöffnet ist. In der folgenden Anweisung wird davon ausgegangen, dass die Abfrage, die Sie umbenennen möchten, geschlossen ist.  
  
### So benennen Sie eine Abfrage um  
  
1.  Aus der **Ansicht** Menü wählen **Projektmappen-Explorer**.  
  
2.  Mit der rechten Maustaste im Projektmappen-Explorer\-Klicken Sie auf die Abfrage, die Sie umbenennen möchten und klicken Sie auf **Umbenennen** im Kontextmenü, das angezeigt wird.  
  
3.  Geben Sie einen neuen Namen für die Abfrage ein, und drücken Sie dann die EINGABETASTE.
