# Microsoft PHP Driver for SQL Server
Der SQL Server-Treiber für PHP wurde entwickelt, um PHP-Anwendungen die zuverlässige und skalierbare Integration mit SQL Server zu ermöglichen. Der SQL Server-Treiber für PHP ist eine PHP 5-Erweiterung, die das Lesen und Schreiben von SQL Server-Daten innerhalb von PHP-Skripts ermöglicht. Er stellt Schnittstellen für den Zugriff auf Daten in allen Editionen von SQL Server 2005 und höher \(einschließlich Express-Editionen\) und SQL Azure bereit und nutzt PHP-Funktionen, wie z. B. PHP-Streams für das Lesen und Schreiben großer Objekte.
