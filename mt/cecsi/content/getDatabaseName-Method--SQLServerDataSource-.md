# getDatabaseName-Methode (SQLServerDataSource)
    
## getDatabaseName\-Methode \(SQLServerDataSource\)  
 Gibt den Datenbanknamen zurück.  
  
## Syntax  
  
```  
  
public java.lang.String getDatabaseName()  
```  
  
## Rückgabewert  
 Ein **String** , der den Datenbanknamen oder NULL enthält, sofern kein Wert festgelegt ist.
