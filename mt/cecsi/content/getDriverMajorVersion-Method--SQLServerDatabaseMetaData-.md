# getDriverMajorVersion-Methode (SQLServerDatabaseMetaData)
    
## getDriverMajorVersion\-Methode \(SQLServerDatabaseMetaData\)  
 Ruft die Hauptversionsnummer dieses JDBC\-Treibers ab.  
  
## Syntax  
  
```  
  
public int getDriverMajorVersion()  
```
