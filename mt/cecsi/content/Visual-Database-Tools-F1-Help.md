# Visual Database Tools - F1-Hilfe
In diesen Themen werden die Befehle und Optionen in den Dialogfeldern und Eigenschaftenfenstern für Visual Database Tools-Elemente, wie beispielsweise das Fenster Tabelleneigenschaften und die Registerkarte Spalteneigenschaften behandelt.  
  
Drücken Sie zum Zugreifen auf Dialogfeldthemen F1, wenn ein Dialogfeld geöffnet ist und es fokussiert ist. Klicken Sie zum Zugreifen auf Eigenschaftenfensterthemen auf das Eigenschaftenfenster, und drücken Sie F1.
