# setDescription-Methode (SQLServerDataSource)
    
## setDescription\-Methode \(SQLServerDataSource\)  
 Legt die Beschreibung der Datenquelle fest.  
  
## Syntax  
  
```  
  
public void setDescription(java.lang.String description)  
```  
  
#### Parameter  
 *description*  
  
 Ein **String** mit der Beschreibung.
