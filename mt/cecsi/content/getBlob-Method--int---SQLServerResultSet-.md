# getBlob-Methode (int) (ISQLServerResultSet)
    
## getBlob\-Methode \(int\) \(ISQLServerResultSet\)  
 Ruft den Wert des angegebenen Spaltenindexes in der aktuellen Zeile dieses [SQLServerResultSet](../content/SQLServerResultSet-Class.md)\-Objekts als Blob\-Objekt in der Programmiersprache Java ab.  
  
## Syntax  
  
```  
  
public java.sql.Blob getBlob(int i)  
```  
  
#### Parameter  
 *i*  
  
 Ein Wert vom Typ **int** zum Angeben des Spaltenindexes.

 Ein Blob\-Objekt.
