# first-Methode (ISQLServerResultSet)
    
## first\-Methode \(ISQLServerResultSet\)  
 Versetzt den Cursor in die erste Zeile dieses [SQLServerResultSet](../content/SQLServerResultSet-Class.md)\-Objekts.  
  
## Syntax  
  
```  
  
public boolean first()  
```
